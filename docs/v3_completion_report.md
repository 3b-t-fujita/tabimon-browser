# Tabimon V3 完了報告

> 更新日: 2026-03-30

## 1. 概要

Tabimon Browser の V3 初回リリース対象について、実装・確認・資料整備まで一通り完了しました。

今回の V3 では、見た目の刷新だけでなく、

- 相棒の成長実感
- `ものがたり / そだてる` の分離
- `EXP / きずな / じゅくれん` の 3 軸育成
- V2 セーブからの安全な移行

を成立させることを主目的として進めました。

## 2. 完了した主な実装

### 成長ループ

- ホームに成長サマリーを追加
- リザルトで `EXP / きずな / じゅくれん / 進化通知` を表示
- 仲間詳細で 3 軸を確認できるようにした
- リザルト後にホームへ戻ると成長が反映される

### ステージ分離

- `ものがたり / そだてる` の 2 タブ化
- `そだてる` を `EXP / BOND / SKILL` の 3 カテゴリに整理
- 各カテゴリを `前半 / 後半` に分け、合計 6 ステージ化
- `そだてる` の解放条件を物語ステージ1/2クリアベースに整理

### バトル・イベント

- 被ダメージ時の小さな揺れリアクションを追加
- 敵味方の揺れが完全同時に見えすぎないよう調整
- ノード5イベントで戦闘が発生しないよう調整

### UI・文言

- 子ども向けに漢字・説明量を抑えた
- `Stage -> ステージ`
- `Easy / Normal / Hard -> かんたん / ふつう / むずかしい`
- `たすっ人 / 助っ人 -> おたすけ`
へ統一した

### 保存・移行

- `OwnedMonsterV3` 対応
- `StageMasterV3` 対応
- `DailyRecord` 対応
- V2 -> V3 移行の補完処理を追加
