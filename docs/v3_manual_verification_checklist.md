# Tabimon V3 手動確認チェックリスト

## 推奨確認環境
- 安定確認用 URL: `http://127.0.0.1:4303`
- 主要入口:
  - `/setup`
  - `/home`
  - `/adventure/stages`
  - `/preview/ui`

## 1. 初期設定
- `/setup` を開く
- ワールドを1つ選ぶ
- `ぼうけんを はじめる` を押す
- `/home` に遷移することを確認する

確認ポイント:
- ワールド選択が視覚的に反映される
- ボタン押下後に `/setup?worldId=...` のまま止まらない
- エラーなくホームへ進む

## 2. ホーム
- 相棒カードが表示される
- `ぼうけんへ いく` など主要 CTA が見える
- 成長サマリーに `EXP / きずな` が見える
- 読み込み中のまま止まらない

確認ポイント:
- 初回開始直後にホームが成立する
- 既存セーブで開いても破綻しない

## 3. ステージ選択
- `/adventure/stages` を開く
- `ものがたり` と `そだてる` を切り替える

確認ポイント:
- ガイド文が切り替わる
- `そだてる` で育成用途の文言が見える
- `ミドリの森 ステージ 1` のように日本語の `ステージ` 表記になっている
- 難易度が `かんたん / ふつう / むずかしい` で見える
- 画面崩れがない

## 4. そだてる 6 ステージ
- `そだてる` タブを開く
- `EXP / BOND / SKILL` の前半・後半が見えることを確認する

確認ポイント:
- 合計 6 ステージある
- 前半は物語のいずれかのステージ1クリアで開く
- 後半は物語のいずれかのステージ2クリアで開く
- ステージ説明が短く分かりやすい

## 5. モンスター詳細
- 任意のモンスター詳細を開く

確認ポイント:
- `EXP / きずな / じゅくれん` が見える
- 相棒設定など既存導線が壊れていない

## 6. リザルト
- 冒険後の `/adventure/result` を確認する

確認ポイント:
- EXP セクション
- きずなセクション
- 熟練セクション
- 進化通知
- CTA
が見えること

## 7. 通し確認
- `setup -> home -> stages -> confirm -> play -> battle -> result -> home` を1周する

確認ポイント:
- リザルト後にホームの `EXP / きずな` が更新される
- ノード5イベントのあとに、戦闘が連続しない
- ダメージを受けた対象だけが揺れる
- `おたすけ` 表記が全体でそろっている

## 8. 移行確認
- 既存セーブを使って起動する

確認ポイント:
- セーブが壊れない
- V3 項目が不足なく補完される
- ホームや詳細画面が表示できる
