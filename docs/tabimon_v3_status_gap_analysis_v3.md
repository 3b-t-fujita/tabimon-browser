## 所見

- V3 初回リリースの「成長ループの見える化」は、ホームとリザルトの接続まで含めて概ね成立した
- 一方で `そだてる` は、内部値差分とリザルト文言までは入ったが、戦利品テーブルや候補抽選までカテゴリ専用化はしていない
- そのため、V3 初回としては成立したが、V3.5 では FARM の戦利品設計を次の改善対象にできる
