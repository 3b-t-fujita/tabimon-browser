## H. 既知制約一覧

以下は**仕様内制約**（バグではない）として確認済み。

| 制約 | 内容 |
|---|---|
| QR受取上限→単純拒否 | QR で仲間/助っ人が上限の場合、入替画面へ行かず拒否メッセージのみ表示 |
| 候補受取上限→入替導線あり | 冒険後の候補受取が仲間上限の場合のみ入替画面に遷移（QR とは別フロー） |
| QR見送り→履歴なし | 「見送る」を選択した場合、同じ QR を再度読み取れる（履歴更新しない） |
| QR生成はpre-selectなし | 仲間詳細から `/qr/generate` に遷移しても、モンスターは一覧から選び直し |
| カメラ直接撮影非対応 | QR 読取はファイル選択（画像アップロード）のみ対応。カメラ API は未実装 |
| 助っ人からQR生成不可 | QR 生成は仲間モンスターのみ対象 |
| 戦闘UI最小限 | HP バー・ログ・スキルボタンのみ。アニメーション・BGM なし |
| マスタ内容固定 | monsters.json / stages.json 等は静的ファイル。ガチャ・更新機能なし |
